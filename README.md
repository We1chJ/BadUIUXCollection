# Bad UI/UX Collection

A curated gallery of intentionally terrible UI and UX design — crafted with care to be as painful as possible.

## What's here

Each subdirectory is a self-contained demo committing a specific design crime. Open `index.html` (or run the local dev server) to experience the horror firsthand.

## Structure

```
/piece-name/
  index.html     # the experience
  README.md      # the crime report
```

## Contributing a piece

1. Create a new directory with a descriptive name (e.g., `invisible-submit-button`).
2. Build the demo — standalone HTML/CSS/JS preferred, but any stack is fine.
3. Add a short `README.md` inside naming the specific UX sin(s) being committed.

The bar: it should be *skillfully* bad, not just broken. Commit to the bit.
